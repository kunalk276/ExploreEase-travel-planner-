### Prerequisites

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/your-username/yatra-travel-app.git
   ```

2. Navigate to the project directory:
   ```
   cd ExploreEase
   ```

3. Install dependencies:
   ```
   npm install
   ```
